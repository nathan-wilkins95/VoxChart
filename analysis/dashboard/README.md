# VoxChart ROI Dashboard

Interactive Dash web dashboard for exploring VoxChart cost-efficiency scenarios.

## Run the dashboard

```bash
pip install -r requirements.txt
python dashboard/app.py
```

Then open http://localhost:8050 in your browser.

## Features

- **Live sliders and dropdowns** — adjust unit size, savings scenario, and VoxChart cost
- **4 KPI cards** — annual OT savings, net ROI, ROI multiple, 5-year cumulative
- **5 interactive charts** — all update in real time as you adjust inputs
- **Export-ready** — charts can be downloaded as PNG via the Plotly toolbar
